# הוראות הפעלה קצרות

1. פתחו את `MMDP_V16_Colab_Focused_Metrics.ipynb` ב-Google Colab.
2. הריצו את שני תאי ההכנה הראשונים.
3. כשהמחברת מבקשת קובץ, העלו את ZIP של V16.
4. הריצו את התא של המפה הרצויה: קלה, בינונית או קשה.
5. כל תא כולל מפה אחת ו-24 conditions.
6. ההתקדמות מוצגת בזמן אמת.
7. התוצאות נשמרות ב-`/content/MMDP_OUTPUT/MMDP_results.csv`.
8. אין הורדה אוטומטית. מורידים ידנית מחלונית Files רק בסיום.
9. הפעלה חוזרת מדלגת על run IDs שכבר הושלמו ב-CSV.
