# הוראות הפעלה קצרות

1. פתחו את `MMDP_V16_Colab_One_Map_Per_Level.ipynb` ב-Google Colab.
2. הריצו את תאי ההכנה הראשונים (שכפול המאגר והתקנת החבילה).
3. הריצו את התא של המפה הרצויה: קלה, בינונית או קשה.
4. כל תא כולל מפה אחת ו-24 conditions.
5. ההתקדמות מוצגת בזמן אמת.
6. התוצאות נשמרות ב-`/content/MMDP_OUTPUT/MMDP_results.csv`.
7. אין הורדה אוטומטית. מורידים ידנית מחלונית Files רק בסיום.
8. הפעלה חוזרת מדלגת על run IDs שכבר הושלמו ב-CSV.
